# Online Retail Data Analysis & Visualization

**Personal Project | May – July 2026**

## Project Objective
Analyze a large retail transactional dataset to uncover revenue trends, customer behavior, return patterns, and high-value customer segments to support data-driven business decisions.

## Dataset
- **Source**: UCI Online Retail Dataset
- **Size**: 1M+ rows of transactional data
- **Period**: 2009 Dec – 2011 Dec

## Tools & Technologies Used
- **SQL/R** — Data cleaning, aggregation, and exploratory analysis
- **Power BI** — Interactive dashboard development (Power Query, DAX, Data Modeling)
- **RFM Analysis** — Customer segmentation

## Key Features of the Dashboard
- Revenue & Return Rate KPI cards (£18.85M total revenue, 4.93% return rate)
- Geographic sales distribution (bubble map)
- Revenue decomposition tree by region and product
- RFM Customer Segmentation
- Return pattern analysis and inventory insights

## Key Insights & Business Recommendations
- Identified a small group of VIP customers contributing disproportionately high revenue
- Detected seasonal return patterns that can help optimize inventory and reduce return costs
- Recommended targeted marketing strategies for High Value and Regular segments
